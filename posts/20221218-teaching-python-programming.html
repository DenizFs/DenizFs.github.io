Teaching Python Programming

2022-12-18

With the experience from my last interpreter project in Python, I decided to teach Python programming to my friends, because in this age almost everyone can benefit from knowing a programming language in some way. Due to the time constraints we all had because of school, we had difficulty meeting for the classes. Still, I can say that it went well because at least they grasped the basics of Python and got a taste of programming. Here is the curriculum we followed:

  1. Variables
  2. Strings
  3. Lists
  4. Loops
  5. Operators
  6. Fibonacci sequence
  7. If-then-else
  8. Dictionaries
  9. Input/output
  10. Functions

I did not go into too much detail with classes and many other aspects of Python so as not to bore them, but I explained the most useful built-in functions and asked them to solve some simple problems to make sure they understood the contents of the lessons.
